Hugo DAY 1 challenge

By Kris 06 July 2021

Github 是什麼？而 Github Pages 又是什麼？

GitHub 是一個透過 Git 進行版本控制的 code 放置平台

Github 就是把一堆使用 Git 版本控制系統的程式碼

Github Pages是把你上傳的html程式碼轉換成網站

Github Desktop 是讓你更方便的上傳程式碼到Github上

  • 每一次存檔都是一個 commit
  1. commit to main 在Desktop存檔
  2. publish branch 重Desktop存到Github上

VScode 是一個編輯器

Hugo 是一個可以快速產生程式碼的工具

hugo 簡介

A Framework, 一種框架工具

It's open-source, 開放原始碼

A static site generator written in Go, 靜態網頁生成器，或稱靜態站點產生器，用 Go 寫的

Build a site easily and quickly by yourself, 你可以輕鬆迅速的構建自己的網站

Hugo 是基於 JAM stack 來建置網站的

themes 是一個有許多可以瀏覽個人網站版型主題的素材庫

此次課程用的主題theme為 liva-hugo
