Hugo DAY 4 challenge

By Kris 21 July 2021

commit 更新網頁資料

當程式修改完成

在cmd 命令提示字元上先終止hugo server [ctrl+C \ 關閉cmd 頁面]

在該儲存檔案的資料夾上執行cmd命令提示字元，進入後執行 hugo

在資料夾下會產生一【public】的資料夾，將資料夾下的檔案複製並貼到(取代舊有內容) github Desktop 的資料夾中，

開啟 github desktop 應用程式，

在左下角點選commit to main (在本地存檔)，

在上方點選publish branch (存到github 上)

回到google chrome 瀏覽器

在網址列輸入 你的網站.github.io

此次存檔更新的部分就都彙編更到網站上了

注意!!!!

每一次要要發布新的資訊\文章等部分都要執行此步驟。

添加評論功能

github pages 為靜態網頁

評論功能是需要後端資料庫來儲存的

所以需要另外來安裝外掛元件

在hugo網頁上搜尋disqus

去申請一的disqus 的帳號

將申請的帳號貼回到 VScode

在config.toml 下找disqusshortname 貼上

如果不是使用Liva-hugo 主題

可能需要另外找該主題使用的位置或自行添加

==================

添加google Analytics 分析

帳戶設定 github pages

資源設定 自己的網站網址.github.io

將產生的GA ID 貼回到config.toml 檔中

如果使用的主題沒有GA 的設定檔

在hugo 網頁上 搜尋 google Analytics

在vs code > layouts > partials >head.html 貼上{{ template ”_internal/google_analytic.html”.}}

在config.toml 貼上googleAnalytics =”申請的GA-ID”

完成後就可以發佈到github pages上
